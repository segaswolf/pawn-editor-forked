=== RELEASE NOTES ===

## What's new in v3d7

### ✅ Facial Animations compatibility
Face type, eyeball color, brow, lid, mouth, skin, and head controllers all copy correctly on duplicate and blueprint load. FA Genetic Heads timing issue fixed — face data survives the SetAllGraphicsDirty override.

### ✅ Bug fixes
- **Faction leaderless bug**: Replacing an NPC faction leader via blueprint now correctly assigns the new pawn as leader
- **Giant gene visual offset** (VE Hussar / VFE-Deserters): Draw offset applies immediately after load — no need to reload or re-apply the xenotype
- **ListingMenu_Items crash**: TypeInitializationException from mods with null style entries now handled gracefully

### ✅ Blueprint system expanded
Save/load now covers: relations, work priorities, inventory, royal titles, records, and active mod list. Full CopyDup support for all new modules.

### ✅ Hotkey picker
Custom editor hotkey in mod settings, persists across sessions.
